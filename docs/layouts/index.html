glas は、Gitインデックスの状況を考慮し、ファイルの変更・追跡ステータスをカラフルに表示するRust製の ls 代替コマンドです。 eza の設計思想を受け継ぎ、開発の生産性を向上させます。

特徴

  • Gitのステージング、削除、未追跡、無視ファイル等のステータスを即座に視覚化
  • フラット表示や特定のGit状態のフィルタリングに対応
  • Rust製かつクロスプラットフォームビルド対応

クイックスタート

インストール：

git clone https://github.com/hagatasdelus/glas.git
cd glas
cargo install --path .
使い方を見る
