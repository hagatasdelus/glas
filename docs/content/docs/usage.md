`glas` は、Gitインデックスの情報を反映しつつファイルをリストアップするコマンドです。

## インストール方法

ソースコードからビルドしてインストールします。

```bash
git clone https://github.com/hagatasdelus/glas.git
cd glas
cargo install --path .
```

---

## 基本的な使い方

カレントディレクトリ内のファイルをリストします。Gitリポジトリ内であれば、各ファイルの状態に応じたステータスカラーが表示されます。

```bash
glas
```

### 主なGitフィルタオプション

`glas` はGitの状態に基づいて表示ファイルをフィルタリングする以下のオプションを備えています：

- `--cached`: Gitで追跡中（インデックスに登録済み）のファイルのみを表示します。
- `--stage`: ステージング環境の情報（ファイルモード、ハッシュ値、ステージ番号等）を `git ls-files --stage` のように出力します。
- `--deleted`: 削除されたファイルのみを表示します。
- `--modified`: 変更が加えられたファイルのみを表示します。
- `--others`: 未追跡のファイルのみを表示します。
- `--ignored`: 無視されたファイルのみを表示します。
- `--include-ignored`: 無視されたファイルも他のファイルと一緒に含めて表示します。

---

## 表示レイアウトの変更

- `-l` (ロング形式): 詳細情報を表形式で出力します。Gitステータス（M, A, Dなど）が左側に左寄せで表示されます。
- `--flatten=all` または `--flatten=<depth>`: ディレクトリ構造を平坦化し、サブディレクトリ内のファイルもまとめてリスト表示します。
- `-f` または `--only-files`: ディレクトリ自体の表示を抑止し、ファイルのみを出力します。
- `-D` または `--only-dirs`: ディレクトリのみを表示します。

> [!TIP]
> `-f` と `-D` を同時に指定した場合は、ファイルとディレクトリの両方を表示（和集合）します。
